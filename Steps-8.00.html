Part Five: Create ODS Layer 8

Creating ODS Database, tables and Connection.

  1. Create the Data base and ODS Tables Use (the attached SQL File)
  2. Verify the tables and database are  created
  3. Create New connection Manager to The ODS database
  4. Select OLEDB Connection
  5. Select new, the server's name and the ODS database Then test connection
  6. Create a new package to load ODS




Loading Dictionary Tables

  1. Drag and drop data flow task And these transformation inside the data flow
  2. Select Source
  3. Select the Staging DB and the Cleaned staging tables
  4. Verify the Columns
  5. Select the Data conversion 
  6. Select all the tables and use this image to map the Datatypes
  7. Select the Destination
  8.  Select the ODS DB and State with the Dic_APP tables
  9. Map the Converted data to the destination columns
  10. Rename the task and run the package once it works then truncate the tables
  11. Select the table in the ODS and verify the data was loaded
  12. Copy and paste the completed task For the next ODS tables
  13. Select the destination and choose the next table
  14. Map the Converted data to the destination columns
  15. Rename the task
  16. Repeat the same steps to configure the rest of ODS tables

  17. Drag and drop the sequence container And place all the task in the container then connecting them

  18. Run the package and verify all the data has been loaded
