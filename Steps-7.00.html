Part Four: Cleaning Data 7

Cleaning Data: Configuring Source and Sort transformation

  1. Drag and Drop these tools Form the SSIS Tool Box
  2. Create a new package In the solution explorer
  3. Select the Staging data base source.
  4. Select the staging table.
  5. Verify the mappings are correct
  6. Connect the source to the sort transformation.
  7. Choose the license number column to sort by and in Ascending order.
  8. Select the remove duplicates box then click okay.
  9. Connect Sort transformation to Conditional split transformation.





Cleaning Data: Configuring Conditional Split

  1. Select the conditional split transformation.
  2. Create the name for the output and copy the condition under the condition box and rename the default output name.
  3. Connect the conditional split to the two destination one for the null check and not null records
  4. Connect the Not null and Null check to the 2 destinations.


Cleaning Data: Configuring Destination

  1. Select the destination for the not null records
  2. Select the connection staging database
  3. Select new to create the table for the clean records
  4. Rename the name of the cleaned staging table
  5. Verify the mapping is correct
  6. Select the Null check destination then Select the connection staging database
  7. Select new to create the table for the clean records
  8. Select new to create the table for the Dirty records
  9. Verify the mapping is correct
  10. Run the Package
  11. Select the tables in SSMS to verify the loaded data
