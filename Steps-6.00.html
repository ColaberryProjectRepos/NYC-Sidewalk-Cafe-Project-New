Parth Three: Staging Layer 6

Staging Layer

  1. Open SSMS
  2. Open New Query
  3. Use the script to create The Data base And  the staging tables and Run the Query.
  4. Open SSIS in Visual Studio
  5. Select File, New and Projects
  6. Select integration services then
  7. Select Integration service project
  8. Select File location and Rename the projects
  9. This will show in the solution explorer



Staging Layer: Connection Manager Configuration

  1. Click connection manager and new connection
  2. Select OLDEB Connection
  3. Select new
  4. Choose Server
  5. Choose the staging Database
  6. Test the connection and hit Okay for each window
  7. This will down here




Configuring Execute SQL task 

  1. Drag and drop Execute SQL task
  2. Select Connection
  3. Select the Database
  4. Select ellipses
  5. Insert Truncate Statement
  6. Rename the Execute SQL Task





Staging Layer : Configuring Source

  1. Drag and drop data Flow Task
  2. Inside the Data Flow Task Drag and drag OLEDB Source and Destination
  3. Select staging connection manager
  4. Select SQL Command to bring in The source
  5. USING SSMS Script to Parse JSON for OLEDB Source(Download in Data File)
  6. Select the File path to insert into script
  7. Determine Data Types of the CSV column
  8. Copy this Script
  9. Paste the SQL Query in the SQL Command test box
  10. Preview the Data
  11. Click Okay



Staging Layer : Configuring Destination

  1. Select OLEDB Destination
  2. Select the Staging DB and the Staging Table
  3. Verify the mapping
  4. Run the Package
  5. Verify the package ran correctly
  6. Select the Tables to view loaded data.
