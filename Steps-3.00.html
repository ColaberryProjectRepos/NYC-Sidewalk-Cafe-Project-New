ETL Life Cycle of NYC Sidewalk Cafe Project 3

The ETL process for this project begins with the downloading data from the source, Converting it into a JSON Files format. This file is then Loaded into the staging database to view the data and begin preliminary Data Profiling. Once the data has been profiled its Transformed by Cleaning and cleansing the data to be inserted into the ODS data base. After the data is cleaned its then profiled to determined what columns can be facts and dimension then loaded into the data warehouse.
